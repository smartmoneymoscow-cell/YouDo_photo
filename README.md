# YouDo Photo

ИИ-инструмент для автоматического отбора и пакетной обработки интерьерных фотографий.

## Возможности

- **Загрузка** CR3/RAW, JPG-эталонов и видео квартиры
- **ИИ-модерация** — автоматический отбор лучших кадров по резкости, экспозиции, композиции
- **Цветокоррекция** — передача цветов по эталонам и видео
- **Пакетный экспорт** — готовые JPG без изменения интерьера

## Интерфейс

4-шаговый мастер:

1. **Загрузка** — исходники, эталоны, видео
2. **Параметры** — критерии отбора, настройки цвета, ограничения
3. **Модерация** — галерея с оценками, ручная проверка, фильтры
4. **Экспорт** — формат, качество, разрешение, именование, ZIP

## Стек

- Чистый HTML / CSS / JS (без фреймворков)
- GitHub Pages для хостинга
- [planned] RAW-конвертация через WASM
- [planned] ИИ-оценка качества кадров

## Запуск локально

```bash
# Любой HTTP-сервер, например:
python3 -m http.server 8000
# или
npx serve .
```

Открыть `http://localhost:8000`
